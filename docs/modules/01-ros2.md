## 1. Why ROS 2? (The Big Picture)

| Feature                  | ROS 1                          | ROS 2 (Humble/Iron/Jazzy/Kilt)         |
|--------------------------|--------------------------------|----------------------------------------|
| Communication Middleware | Custom TCP/UDP                 | DDS (Data Distribution Service)        |
| Real-time & Safety       | Not guaranteed                 | Deterministic, QoS policies            |
| Multi-robot              | Painful                        | Built-in (Domains, Security)           |
| Windows & Embedded       | Very limited                   | Full support                           |
| Security                 | None                           | SROS2 (DDS-Security)                   |
| Lifecycle Management    | No concept                     | Managed nodes                          |

> **Real-world users**: NASA, Bosch, Toyota, Intel, NVIDIA Jetson, PAL Robotics, Foxglove

## 3. Complete Development Environment Setup

### 3.1 Ubuntu 24.04 + ROS 2 Jazzy (Step-by-step 2025)

```bash
# Locale fix
sudo apt update && sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

# Add ROS 2 apt repository
sudo apt install software-properties-common
sudo add-apt-repository universe
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# Install ROS 2 Jazzy Desktop
sudo apt update
sudo apt install ros-jazzy-desktop python3-colcon-common-extensions

# Source ROS 2 in every terminal
echo "source /opt/ros/jazzy/setup.bash" >> ~/.bashrc
source ~/.bashrc
